# System Settings

The **System Settings** area allows administrators to configure global behavior and preferences for the Recon AI organization.

## General Settings

This section controls the fundamental properties of your tenant.

![General Settings](../assets/admin_settings_general.png)

*   **Organization Name**: The display name of your entity (e.g., *test company*).
*   **Timezone**: The operational time zone used for logs, schedules, and reports (e.g., *UTC-09:00*).
*   **Business Rules**: The default logic set applied to reconciliation (e.g., *Standard*).
*   **Default Currency**: The primary currency for financial calculations (e.g., *INR - Indian Rupee*).

## Notification Settings

Manage how and when the system communicates with users.

![Notification Settings](../assets/admin_settings_notifications.png)

You can toggle the following alerts on or off:

*   **Email Notifications**: Enable or disable all outbound emails.
*   **System Alerts**: Notifications regarding critical system errors or warnings.
*   **Reconciliation Alerts**: Updates on the status of reconciliation jobs (e.g., completion, failure).
*   **Maintenance Notifications**: Alerts about scheduled downtime or updates.
