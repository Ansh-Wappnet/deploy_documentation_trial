# SFTP Server

The **SFTP Server** section allows you to manage secure file transfer connections for automated data retrieval. This is ideal for batch processing large volumes of transaction data on a scheduled basis.

![SFTP Connections](../assets/sftp_setup.png)

## Connection Details
Each configured connection displays key information:

*   **Host/Address**: The address of the SFTP server (e.g., `0.tcp.in.ngrok.io`).
*   **Port**: The connection port (e.g., `14428`).
*   **Remote Path**: The directory on the server where files are located.
*   **Status**: Last sync time and connection health.

## Actions

*   **Test**: Verify connectivity with the server.
*   **Edit**: specific configuration details.
*   **Delete**: Remove the connection configuration.
