# Overview

The **Super Admin Panel** is the centralized management interface for the Recon AI reconciliation platform. It serves as the nerve center for platform operators, enabling them to control every aspect of the multi-tenant environment.

## Purpose

The primary goal of this panel is to allow superusers to:

- **Configure System-Wide Settings:** Manage global parameters that affect all tenants.
- **Manage Tenants:** Onboard, approve, and configure client organizations.
- **Control Access:** Define roles and permissions to ensure secure operations.
- **Define Logic:** Create and maintain the reusable rules that drive reconciliation.
- **Oversee Subscriptions:** Manage billing tiers and feature access for clients.

## Intended Users

This documentation is specifically written for:

- **Super Admins:** Users with full system access.
- **Platform Operators:** Technical staff managing the daily health of the platform.
- **Internal Support & Ops Teams:** Staff responsible for onboarding and troubleshooting client issues.

## Navigation Structure

The admin panel is organized into logical sections via the left-hand **Sidebar Menu**. The menu is visually split into "header" sections (gray text) and "links" (white text).



### Menu Sections

1.  **Dashboard**: The main landing page.
2.  **Clients Management**:
    - **Clients**: Manage tenant organizations.
    - **Domains**: Manage URL entry points.
3.  **Base Configuration**:
    - **Modules**: System capabilities.
    - **Permissions**: Access control definitions.
    - **Rule Components**: Reusable logic blocks.
    - **Subscription Plans**: Commercial tier definitions.
